# Lotto Number Generator

실시간 로또 번호 생성기 및 제휴 문의 페이지입니다.

## 주요 기능
- **번호 생성**: 1~45 사이의 중복 없는 6개 번호를 생성하고 정렬하여 표시합니다.
- **테마 모드**: 다크 모드와 라이트 모드를 지원하며, 사용자의 설정을 기억합니다.
- **제휴 문의**: Formspree를 통한 실시간 메일 문의 기능을 제공합니다.
