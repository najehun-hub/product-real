# AI Tools - Gender Test & Lotto Generator

Teachable Machine을 이용한 성별 판별 AI 테스트와 로또 번호 생성기를 제공하는 통합 도구 사이트입니다.

## 주요 기능
### 1. 성별 AI 테스트 (Gender AI Test)
- **AI 분석**: Google Teachable Machine 모델을 사용하여 사진 속 인물의 성별 관상을 분석합니다.
- **다양한 입력 방식**: 
  - 사진 업로드 (파일 선택)
  - 드래그 앤 드롭 (Drag & Drop)
  - 실시간 웹캠 촬영 (Webcam Capture)
- **시각화 결과**: 분석 결과를 퍼센트 바와 함께 직관적으로 표시합니다.

### 2. 로또 번호 생성기 (Lotto Generator)
- 1~45 사이의 중복 없는 6개 번호를 생성합니다.
- 번호별 공식 색상 테마를 적용하여 시각화합니다.
- 생성된 번호를 오름차순으로 자동 정렬합니다.

### 3. 공통 기능
- **다크/라이트 모드**: 사용자의 선호에 맞는 테마 전환 및 설정 유지.
- **반응형 디자인**: 모바일 및 데스크탑 환경 최적화.
- **소통 기능**: Disqus 댓글 시스템 및 Formspree 제휴 문의 폼 통합.

## 기술 스택
- **AI**: Teachable Machine, TensorFlow.js
- **Frontend**: HTML5, CSS3 (Custom Variables, Flexbox, Animations), JavaScript (ES6+)
- **Service**: GitHub Pages, Formspree, Disqus
